# install Free Bootstrap 3 Admin Template  for mac
----------------------------

##前置条件
1. npm 
2. bower
3. gulp

###前置条件安装
npm 安装  
```
brew install npm
npm -v //版本验证
```
power 安装  
```
npm install bower 
bower -v
```
gulp 安装  
```
npm install gulp -g
gulp -v
```
📣📣📣： 如果已经安装了则无需在安装

###软件依赖安装
####下载源码
```
git clone https://github.com/puikinsh/gentelella.git
cd gentelella
```
或者直接
```
wget https://github.com/puikinsh/gentelella/archive/master.zip
cd gentelella
```
####依赖安装并构建js
```
npm install 
gulp
```
📣📣依赖安装会自动加载当前目录下的package.json文件
